# README

## 開発言語

- Ruby 2.6.5
- Ruby on Rails 5.2.5


## 就業Termの技術

- お気に入り機能
- フォロー機能


## カリキュラム外の技術

- redcarpet
- coderay


## 実行手順

```
$ git clone git@github.com:grohno/tabishinan.git
$ cd tabishinan
$ bundle install
$ rails db:create db:migrate
$ rails s
```
